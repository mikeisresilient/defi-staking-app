# Asset Inventory - Frontend

This document provides an inventory of the static assets and visual elements used by the Frontend (web) component of the DeFi Staking App.

## 1. Images

These are static image files used for branding, iconography, and token representation.

*   **`src/bank.png`**:
    *   **Purpose**: Likely used as an icon or logo representing the "bank" or staking platform within the application.
    *   **Type**: PNG image.

*   **`src/eth-logo.png`**:
    *   **Purpose**: The official Ethereum logo, used to represent the underlying blockchain technology.
    *   **Type**: PNG image.

*   **`src/logo.png`**:
    *   **Purpose**: The primary logo for the DeFi Staking App itself.
    *   **Type**: PNG image.

*   **`src/tether.png`**:
    *   **Purpose**: The logo or icon for the Tether (mUSDT) token, used to visually identify the staked asset.
    *   **Type**: PNG image.

*   **`src/token-logo.png`**:
    *   **Purpose**: A generic token logo, potentially used as a placeholder or for other tokens within the application.
    *   **Type**: PNG image.

*   **`public/favicon.ico`**:
    *   **Purpose**: The favicon (favorite icon) for the website, displayed in browser tabs and bookmarks.
    *   **Type**: ICO image.

## 2. Animations

The application incorporates dynamic visual effects for an enhanced user experience.

*   **Particle Effects**:
    *   **Mechanism**: Configured via `src/components/ParticleSettings.js` and rendered using libraries like `react-particles-js` or `react-tsparticles`.
    *   **Purpose**: Provides a dynamic and engaging background animation, contributing to the overall aesthetic of the application.
    *   **Type**: JavaScript-driven visual effect.
